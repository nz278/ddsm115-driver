# DDSM115 Library
Python interface for controlling DDSM115 motor

## Features
1. Set and query motor ID
2. Set modes
3. Send speed, current/torque, position commands
4. Receive motor feedback
5. Control individual motors or multiple simultaneously

## Installation
pip install -r requirements.txt
