# DDSM115 Library
Python library for controlling Waveshare DDSM115 servo motors over RS485 serial communication.

## Features
- Set and query motor ID
- Set modes
- Send speed, current/torque, position commands
- Receive motor feedback
- Control individual motors or multiple over shared bus

## Installation
pip install -r requirements.txt  
See `examples/basic_test.py`
